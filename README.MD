```Задание```
Необходимо создать модель вет клиники, используя объектно-ориентированный язык
программирования (Java, Python, C#):
1. Создать классы для различных животных (домашние, дикие). При создании классов
   продемонстрировать использовать интерфейсов и абстрактных классов.
2. Создать класс для ветеринара, который будет «лечить» поступающее животное:
   a. выводить в консоль сообщение, что началось лечение животного с кличкой
   b. вызывать у животного метод makeNoise()
   c. выводить в консоль сообщение, что лечение закончилось
3. В основном main методе программы создать объект ветеринара, создать очередь из
   разных животных и «вылечить» всех животных
   В качестве решенного задания ожидаем проект в гите.
